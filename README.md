# ChestRandomizer

ChestRandomizer is a Minecraft plugin supporting versions 1.16.x that updates the chests you select with the items of your choice.

## Installation

Use the [Spigot page](https://www.spigotmc.org/resources/chestrandomizer.455015/) to find and download the most stable release of this plugin. Install into `plugins` directory of your Minecraft server.

## Usage

Type `/cr` on your Minecraft server to output the command skeleton. For more detailed instructions, check the [Wiki](https://github.com/abstractionAlpha/ChestRandomizer/wiki).
